# 【なぜこのアプリを作ったか】 
私自身筋トレが好きで、法人/個人共にメディアの運用経験があることからトレーニングジム紹介メディアを連想し作成しました。

# 機能一覧
* メタタグを用いたSEO最適化
* パンくずリスト
* 同一IPからの過剰なアクセスの制限（厳し目に5分間に50回以上）
* ログイン機能
* コメント機能
* いいね機能
* 画像複数投稿機能
* ブログ投稿機能
* 関連投稿表示機能
* gemを用いた管理画面
* ページネーション機能
* 問い合わせ機能（内容確認機能）
